# Планировщик заданий cron
> Кронос - бог времени

Запланировать задачу
```crontab -e```

Посмотреть задачи
```crontab -l```

в файле можно указывать 
- точное значение 3
- инервал 2-4
- список 1, 3, 5
- любой `*`
- периодичность */2
- cочетание 3,5,6-12,*/2

0-7 - дни недели. 1 - пн, 0,7 - воскресенье
